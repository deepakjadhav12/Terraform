In this task, I used Terraform to deploy a secure Ubuntu EC2 instance in AWS. The instance was launched in the default VPC using the latest official Ubuntu AMI available in the us-east-1 region. I configured the security group to allow incoming traffic on ports 22 (SSH) and 443 (HTTPS) to ensure secure remote access and encrypted web traffic. The instance was provisioned as a t2.micro, and key-pair authentication was enabled to allow secure SSH login.
